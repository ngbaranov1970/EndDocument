# EndDocument Frontend

Frontend на Vue 3 + Vite для работы с документами и организациями.

## Маршруты

- `/` - список документов, сгруппированных по организациям
- `/create` - создание документа
- `/organizations/create` - добавление организации

## Быстрый запуск

```bash
npm install
npm run dev
```

## Проверка сборки

```bash
npm run build
```
